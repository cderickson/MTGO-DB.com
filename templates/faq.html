I already use MTGO-Tracker Desktop. Do I have to re-do all my manual revisions?

No! You can import your saved data from MTGO-Tracker.

The preferred method to accomplish this is to:

1

Upload files normally

Use Import GameLogs to upload your MTGO files first

2

Import your revisions

Use Import from MTGO-Tracker to apply your manual revisions

Why do I need to upload logs as a .zip file? Can I upload individual files?

File timestamps are crucial: GameLog files don't include timestamp data in their contents.

The parsing process relies on file modification date to record when a match was played. This metadata is:

  • Not preserved when individual files are uploaded
  • Preserved when zipped files are uploaded

This is why we require the .zip format to maintain accurate match timing data.

What is the Zip-MTGO-Files script doing?

The script performs these actions:

1

Scan your PC

Looks for files matching MTGO Game/DraftLog naming conventions and file types

2

Copy files

Creates copies of found Game/DraftLog files

3

Create zip archive

Adds all files to a .zip that can be uploaded to MTGO-DB

View Script Source Code

The script code is open source and available on GitHub

How long does data processing take?

Processing time depends on several factors:

File Count

More files = longer processing

Game Complexity

Longer games take more time

Server Load

Peak times may be slower

Email notification: You'll receive a Load Report via email when processing is complete, so you don't need to wait around!

Is my MTGO data secure and private?

Your data is private: Only you can access your uploaded MTGO data.

MTGO-DB takes data privacy seriously:

  • Account-based access: Data is tied to your registered account
  • No data sharing: Your game data is never shared with other users
  • Secure storage: Data is stored securely and encrypted
  • Data control: You can export or delete your data at any time
