1 I already use MTGO-Tracker Desktop. Do I have to re-do all my manual revisions?

No! You can import your save data from MTGO-Tracker.

The preferred method to accomplish this is to:

1
Upload files normally

Use Import GameLogs to upload your MTGO files first


2
Import your revisions

Use Load Revisions to apply your manual revisions

2 Why do I need to upload logs as a .zip file? Can I upload individual files?

Zip files are required to maintain accurate match dates and times.

File Timestamps

GameLog files don't include timestamp data in their contents.

  • Our data ingestion process uses file modification date to record the date and time a match was played.
  • This metadata is not preserved when individual files are uploaded.
  • File modification date is preserved when zipped files are uploaded.

3 What is the Zip-MTGO-Files script doing?

The script performs these actions:

1
Scans your PC

Looks for files matching MTGO Game/DraftLog naming conventions and file types.


2
Copies files

Creates copies of discovered Game/DraftLog files.


3
Creates .zip archive

Adds all files to a .zip that can be uploaded to MTGO-DB for processing using the Import GameLogs feature.

View Script Source Code

The script code is open source and available on GitHub.
