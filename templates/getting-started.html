Import from GameLogs

Recommended for new users: Use this option if you are importing data for the first time.

1

Download the Zip Script

Download and run the Zip-MTGO-Logs script. A .zip file containing your GameLog and DraftLog files will be generated.

2

Navigate to Import

Click on Processing → Import GameLogs in the navigation menu.

3

Upload Your Files

Navigate to the .zip file created in Step 1 and upload it.

Your browser may ask you to confirm your upload.
4

Process Data

Click Import. Your files will be processed in the background and a Load Report will be emailed to you upon completion.

Import from MTGO-Tracker

For existing MTGO-Tracker users: Use this option if you have previously used the MTGO-Tracker desktop application and would like to upload your saved data.

1

Locate Your Data

Find your MTGO-Tracker database file (usually named mtgo_tracker.db).

2

Navigate to Import

Click on Processing → Import from MTGO-Tracker in the navigation menu.

3

Upload Database

Upload your MTGO-Tracker database file to import your existing data and manual revisions.
