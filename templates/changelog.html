Version 1.0.0 - Initial Release August 17, 2025

Initial Release: The first public release of Mox Data, a web-based MTGO data analytics platform.

Mox Data is a web-based port of MTGO-Tracker, a free and open-source desktop application for analyzing MTGO data.

New Features

  • Cloud-based data storage: Upload and process MTGO GameLog and DraftLog files.
  • Analytics dashboards: New dashboards to view match statistics and performance metrics.

Planned Features: We're constantly working on new features and improvements based on user feedback.
