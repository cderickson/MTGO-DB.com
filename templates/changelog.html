Version 1.0.0 - Initial Release November 1, 2023

Welcome to MTGO-DB! The first public release of the web-based MTGO data analytics platform.

Core Features

Data Import

Upload and process MTGO GameLog and DraftLog files

Analytics Dashboard

View match statistics and performance metrics

Data Tables

Browse matches, games, drafts, and plays

Data Export

Export your data as CSV files

Key Capabilities

  • User Registration: Secure account creation and email verification
  • GameLog Processing: Automated parsing of MTGO game files
  • Draft Analysis: Track draft picks and performance
  • Match History: Detailed match and game statistics
  • Data Cleaning: Tools to improve and organize your data
  • MTGO-Tracker Import: Migrate data from the desktop application
