    Automatically assign deck names based on card analysis

    Choose which matches to analyze for deck name suggestions

    How it Works

    This feature analyzes your deck compositions and automatically suggests appropriate deck names based on card patterns, colors, and archetypes.

    Important Note

    This process may take a few minutes to complete depending on the number of matches to analyze. You can continue using other features while it runs in the background.

    Import from MTGO-Tracker (Windows)

    Select the ALL_DATA, DRAFTS_TABLE, and PICKS_TABLE files from your MTGO-Tracker/save/ folder

    How to Select Files

    1. Navigate to your MTGO-Tracker/save/ folder
    2. Hold Ctrl (or Cmd on Mac) and select these files:
      • ALL_DATA
      • DRAFTS_TABLE
      • PICKS_TABLE

    Important Notes

    • Records added will not have an associated raw file archived and will not be able to be re-processed.
    • This will overwrite any manual revisions and is not reversible.

    Re-Process Data

    Your archived GameLog and DraftLog files will be re-parsed. Use this function if MTGO-Tracker parsing rules have changed.

    Any manual revisions you have made (such as deck names) will not be lost.

    Processing Details

    • This may take several minutes depending on how many archived files you have to reprocess.
    • A load report will be emailed upon completion.
