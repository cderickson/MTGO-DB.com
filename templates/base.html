    Import GameLogs/DraftLogs

    Upload your zipped GameLog and DraftLog files for processing

    Select the .zip file created by the Zip-MTGO-Logs script

    Importing Details

    • This may take several minutes depending on how many archived files you have to reprocess.
    • A load report will be emailed upon completion.

    Import from MTGO-Tracker (Windows)

    Upload your MTGO-Tracker save files for processing

    Select the ALL_DATA, DRAFTS_TABLE, and PICKS_TABLE files from your MTGO-Tracker save folder.

    How to Select Files

    1. Navigate to your MTGO-Tracker/save/ folder
    2. Hold Ctrl (or Cmd on Mac) and select these files:
      • ALL_DATA
      • DRAFTS_TABLE
      • PICKS_TABLE

    Warning

    • Records added will not have an associated raw file archived and will not be able to be re-processed.
    • This will overwrite any manual revisions and is not reversible.

    Load Revisions

    Upload user-input revisions from your MTGO-Tracker desktop application

    Select the ALL_DATA file from your MTGO-Tracker save folder.

    How to Select File

    1. Navigate to your MTGO-Tracker/save/ folder
    2. Select this file: ALL_DATA

    Warning

    • This will load user-input revisions from your MTGO-Tracker save file.
    • This process will overwrite existing revisions and is not reversible.

    Re-Process Data

    Delete existing data and re-process your archived files.

    When to use this feature

    • Re-parse your archived GameLog and DraftLog files if MTGO-Tracker parsing rules have changed.
    • Manual revisions you have made (such as deck names) will not be lost.

    Processing Details

    • This may take several minutes depending on how many archived files you have to reprocess.
    • A load report will be emailed upon completion.
